# DjangoStarter 概览

DjangoStarter 是面向现代全栈开发的 Django 项目模板，目标是让你在一个统一的工程里快速搭建后端 API、管理后台与轻前端交互。它结合 Django 的成熟生态与 Django-Ninja 的高性能接口能力，并通过 Tailwind CSS、HTMX、Alpine.js 等前端工具提供高效的交互开发体验。

## 适用场景

- 需要快速搭建中后台或管理系统
- 以 Django 为核心并希望具备现代化前端体验
- 重视开发效率、工程规范与可维护性

## 核心能力

- Django 6 + Django-Ninja API 架构
- Split Settings 配置体系，支持多环境配置
- 内置缓存、监控与安全中间件
- 前端基于 Tailwind CSS + Flowbite
- 预置 CLI 与自动代码生成工具

## 技术栈摘要

- 后端：Django 6 / Django-Ninja / Redis / Simple History
- 前端：Tailwind CSS / Flowbite / HTMX / Alpine.js
- 工具链：uv / pnpm / gulp / Docker
